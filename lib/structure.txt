The original codebase had files spread across:
  lib/app/controllers/    lib/app/modules/*/controllers/
  lib/app/views/          lib/app/modules/*/views/
  lib/app/bindings/       lib/app/modules/*/bindings/
  lib/app/core/           lib/core/          (two separate core trees!)
  lib/app/widgets/        lib/app/modules/*/widgets/

This is the flattened, readable structure. Everything is one level deep
per category. All imports use package:school_app/ absolute paths.

Zero functionality changes. Zero API changes. Zero RBAC changes.
